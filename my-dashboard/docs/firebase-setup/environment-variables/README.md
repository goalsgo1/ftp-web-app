# 환경 변수 (Environment Variables)

> **Firebase 환경 변수 설정**

이 폴더에는 Firebase 기능을 사용하기 위한 환경 변수 설정 방법이 포함되어 있습니다.

---

## 📄 파일

- **FIREBASE_ENV_SETUP.md** - Firebase 환경 변수 설정 가이드

---

## 🎯 목적

Firebase 기능(인증, 데이터베이스 등)을 사용하기 위해 필요한 환경 변수를 설정합니다.

**포함 내용:**
- Firebase Console에서 설정 값 복사
- `.env.local` 파일 생성
- 환경 변수 입력
- 설정 확인 및 문제 해결

---

## ⚠️ 필수 사항

Firebase 기능을 사용하려면 **반드시** 이 설정이 필요합니다.

**영향:**
- 로그인/회원가입 기능
- 데이터 저장/불러오기
- 모든 Firebase 기능
