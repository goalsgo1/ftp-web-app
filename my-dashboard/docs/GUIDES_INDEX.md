# 📚 활성 사용 가이드 인덱스

---

## 🟢 빠른 시작

### QUICK_START.md
**용도**: 개발 서버 실행/종료 빠른 참조  
**대상**: 처음 시작하는 개발자  
**내용**: 기본 실행, 종료, 간단한 문제 해결

**위치**: 루트 디렉토리

---

## 🚀 개발 서버

### DEVELOPMENT_SERVER_GUIDE.md
**용도**: 개발 서버 상세 가이드  
**대상**: 포트 변경, 고급 문제 해결이 필요한 개발자  
**내용**: 포트 변경, 프로세스 강제 종료, 상세 문제 해결

**위치**: 루트 디렉토리

### CHECK_RUN_STATUS.md
**용도**: 실행 가능 상태 점검  
**대상**: 실행 전 상태를 확인하고 싶은 개발자  
**내용**: 현재 상태 확인, 체크리스트, 제한 사항

**위치**: 루트 디렉토리

---

## 🔥 Firebase 설정

### FIREBASE_ENV_SETUP.md
**용도**: Firebase 환경 변수 설정  
**대상**: Firebase 기능을 사용하려는 모든 개발자  
**내용**: `.env.local` 파일 생성, 환경 변수 입력

**위치**: 루트 디렉토리  
**필수**: Firebase 기능 사용 시 필수

### FIREBASE_AUTH_SETUP.md
**용도**: Firebase 인증 활성화  
**대상**: 로그인 기능을 사용하려는 개발자  
**내용**: 이메일/비밀번호 인증 활성화, 테스트 계정 생성

**위치**: 루트 디렉토리  
**필수**: 로그인 기능 사용 시 필수

## 🔍 문서 찾기

### 처음 시작하는 경우
