# 개발 가이드 (Development Guides)

> **프로젝트 개발에 필요한 모든 가이드 문서**

이 폴더에는 개발 서버 실행, 상태 확인, 컴포넌트 사용법 등 개발에 필요한 모든 가이드가 포함되어 있습니다.

---

## 📁 폴더 구조

### 빠른 시작 (quick-start)
**목적**: 개발 서버 실행/종료 빠른 참조 가이드  
**대상**: 처음 시작하는 개발자  
**파일**: `QUICK_START.md`

### 서버 관리 (server-management)
**목적**: 개발 서버 상세 관리 및 고급 설정  
**대상**: 포트 변경, 고급 문제 해결이 필요한 개발자  
**파일**: `DEVELOPMENT_SERVER_GUIDE.md`

### 상태 확인 (status-check)
**목적**: 실행 가능 상태 점검 및 체크리스트  
**대상**: 실행 전 상태를 확인하고 싶은 개발자  
**파일**: `CHECK_RUN_STATUS.md`

### 컴포넌트 사용법 (component-usage)
**목적**: 공통 컴포넌트 사용법 및 개발 가이드  
**대상**: 새 기능을 추가하는 개발자  
**파일**: `DEVELOPMENT_GUIDE.md`

---

## 🚀 빠른 시작

처음 시작하는 경우 다음 순서로 진행하세요:

1. **빠른 시작** → `quick-start/QUICK_START.md`
2. **Firebase 설정** → `../firebase-setup/environment-variables/FIREBASE_ENV_SETUP.md`
3. **컴포넌트 사용법** → `component-usage/DEVELOPMENT_GUIDE.md`
