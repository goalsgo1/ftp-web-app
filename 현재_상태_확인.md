#### 2단계: 초기 설정
```powershell
# Git 사용자 정보 설정
git config --global user.name "Your Name"
git config --global user.email "your.email@example.com"

# 프로젝트 디렉토리로 이동
cd C:\FTP

# Git 저장소 초기화
git init
git branch -M main

# 파일 추가
git add .

# 첫 커밋
git commit -m "Initial commit"
```

### 선택지 2: Git 없이 수동 배포 진행 (지금 바로 가능) 🔧

Git 설치 없이도 배포를 진행할 수 있습니다.

#### 현재 가능한 작업

1. **프로젝트 구조 정리**
   ```powershell
   # 파일을 public 디렉토리로 이동
   Move-Item index.html public\index.html -ErrorAction SilentlyContinue
   Move-Item world_time.html public\world_time.html -ErrorAction SilentlyContinue
   Move-Item disk_info.php public\disk_info.php -ErrorAction SilentlyContinue
   Move-Item gallery.php public\gallery.php -ErrorAction SilentlyContinue
   ```

2. **FTP로 서버에 전송**
   - FTP 클라이언트 사용 (FileZilla 등)
   - 또는 PowerShell의 FTP 명령어 사용

3. **서버에서 수동 배포 스크립트 실행**
   ```bash
   # 리눅스 서버에서
   ~/deploy/manual_deploy.sh
   ```

#### 수동 배포 상세 가이드

`단계별_배포_가이드.md`의 **"9. 수동 배포 방법 (Git 없을 때)"** 섹션 참고
